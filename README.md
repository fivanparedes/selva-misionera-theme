# Selva Misionera Theme

A pair of VS Code color themes inspired by the subtropical jungle of Misiones, Argentina — home to Iguazú Falls, the Yaguaretá jaguar, and the most biodiverse rainforest in Argentina.

Two variants are included:

- **Selva Misionera** — dark theme with deep forest greens, tierra colorada earth tones, and arroyo water blues.
- **Selva Misionera Clara** — light theme with vivid foliage greens, warm amber, and deep teal on a sunlit canopy background.

---

## Color Philosophy

Both themes share a common syntax color hierarchy derived from elements of the Misionera rainforest:

| Role | Dark | Light | Inspiration |
|---|---|---|---|
| Keywords / storage | `#4fa15a` | `#2e7a46` | Jungle undergrowth |
| Functions / methods | `#c9a028` | `#a07820` | Yaguaretá amber |
| Classes / types / namespaces | `#c4905c` | `#9a6835` | Tierra colorada (red earth) |
| Variables | `#8adde8` | `#1f8a96` | Arroyo (stream water) |
| Strings | `#e39a67` | `#b06030` | Bark and resin |
| Numbers / constants | `#dcb170` | `#9a7020` | Dry season foliage |
| Parameters | `#c8a5e0` | `#7a50a0` | Orchid |
| Properties / attributes | `#ec8f65` | `#c06030` | Clay |
| JSON keys | `#7ecfdc` | `#1878a0` | Clear water |
| Comments | `#6b8d72` italic | `#7a9485` italic | Fog |
| Regexp | `#b8df71` | `#5e8c18` | Fern |

---

## Dark Theme — Selva Misionera

Background palette from deepest to lightest:

- Chrome (bars, tabs, terminal, panel): `#08180f` / `#07150d`
- Sidebar: `#0f2a19`
- Editor: `#0e2816`
- Widgets / suggest: `#143521`

## Light Theme — Selva Misionera Clara

Background palette from most saturated (chrome) to lightest (editor):

- Chrome (activity bar, title bar, status bar, panel, terminal): `#7aaa6e`
- Sidebar: `#b8d8aa`
- Editor: `#daf2cc`
- Widgets / suggest: `#c4e4b4`

## Installation

### From the VS Code Marketplace

1. Open VS Code.
2. Go to **Extensions** (`Ctrl+Shift+X` / `Cmd+Shift+X`).
3. Search for **Selva Misionera**.
4. Click **Install**.
5. Open the Command Palette (`Ctrl+Shift+P`), run **Preferences: Color Theme**, and select **Selva Misionera** or **Selva Misionera Clara**.

### From a VSIX file

```bash
code --install-extension selva-misionera-theme-0.2.0.vsix
```
